# Clinic Management System
The Oceana Clinic Management System is developed using Visual Basic .NET.

<br>

## Features
* Login authentication
* Manage user account (Admin, Staff Nurse & Doctor)
* Manage patient info
* Manage prescription
* Manage and generate bill
